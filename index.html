Что такое PLC:
•
Programmable Logic Controller ( PLC )
•
Power Line Communitation ( PLC )
Начнем с PLC модема, как вызывающего больший интерес. Контроллеров якобы уже хватает, хотя предлагаемая платформа контроллеров тоже - оригинальна.
Характеристики модема
Рабочий диапазон питающего напряжения
 DC 0-48V; AC 0-250V
Максимальная потребляемая мощность не превышает
 1Вт
Режим работы модема
 непрерывный; прозрачный
Разъемы
 Arduino UNO совместимый
Используемые интерфейсы один из SPI; I2C; UART
Логическая организация сетей
 простые одноранговые сети без возможности разделения на подсети, каждый модем имеет уникальный адрес
Скорость передачи данных по PLC сети
 1200 - 19200 бит/с
Диапазон частот при передаче данных по PLC
 1.87 MHz
Уровень сигнала
 менее 116 дбмкВ
