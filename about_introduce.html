Target: creation distributed control system, IoT, build-in systems, etc.

The optimal solution must be in the ratio features / cost / quality.
The system consists of modules ( master - slaves ).
Module consists of one or more circuit boards.
Main board based on microcontroller STM32F100; STM32F103.
Interface board contains connectors, functional module elements.
The boards are made functional to be installed in a box on the DIN-rail.
