Linux PLC is a layer L1 module.
Linux PLC
Software Platform:    Embedded Linux


    Hardware
    Processor:    Type 32-bit i.MX53 / i.MX65 processor; ARM Core A8Processor
    Speed:          800 MHz / 1000 MHz
    Memory:        256 MB / 512 MB NOR flash; 256 MB / 512 MB SDRAM





    Additional Population
    Options
    - RJ-45 Ethernet connector
    - 2 serial ports: RS-232, RS-485
    - ADC: 12-bit/8-channel 0-20 mA
    - 6 Relays 220V/10A with current sensors
    - 1-Wire interface ( thermal sensors, key-button, etc. )
    - MicroSD slot
    - 1 port USB
    - HDMI display with touch screen
    - 2 CAN interfaces
    - GSM 2+
    - Differences extended modules


    Environmental
    - Operating Temperature -40 °C to +85 °C; 33% WLAN duty cycle
    - Relative Humidity 5% to 95% (non-condensing)
    - Altitude 12 000 feet (3 658 meters)
